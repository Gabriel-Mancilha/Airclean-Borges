Projeto: Airclean Borges. 
 
Objetivo: 

O projeto consiste em desenvolver um sistema de agendamento para uma empresa com serviços para ar-condicionado, como: instalação, manutenção e limpeza dos equipamentos. 

A empresa enfrentava problemas como a má distribuição de carga horaria para seus funcionários, dificuldade na comunicação entre o setor de atendimento e a equipe técnica, pois o setor de atendimento marcava os horários por meio de grandes planilhas compartilhadas, onde os técnicos poderiam se confundir. E dificuldade no controle de horário dos técnicos e dificuldades para saber os horários disponíveis para os técnicos. 

Com base nesses desafios propostos, optamos na criação de um sistema de agendamento online. O usuário é capaz de selecionar o tipo de serviço (reparo, instalação e limpeza), selecionar dia e horário para aquele serviço, vale lembrar que só será possível o cliente marcar se o dia, horário e serviço tiverem algum técnico disponível. Após isso, o sistema seleciona um funcionário para realizar aquele trabalho e o notifica. 

Após o funcionário estar agendado ele fica inapto para realizar outro serviço naquele mesmo dia/período. O funcionário recebe um cronograma com dias/horas/tipo de serviço/endereços, além disso o sistema não permite que os técnicos trabalhem mais de 6 horas seguidas e nem mais de 8 horas no dia. Seguindo o padrão proposto pela empresa. 

O sistema também tem um chat, onde o cliente pode se comunicar com a equipe de atendimento para tirar dúvidas ou cancelar algum agendamento. 

O sistema guarda todas as informações de cargas horarias dos funcionários e as armazena, para manter um registro caso haja divergência.  

Com a implementação desse sistema, a empresa passa a ter maior controle e organização sobre seus agendamentos, eliminando falhas decorrentes do uso de planilhas manuais e melhorando significativamente a comunicação entre os setores. O sistema garante eficiência na distribuição de tarefas, respeitando as cargas horárias e evitando sobrecarga dos técnicos. 

Dessa forma, o projeto contribui diretamente para o aumento da produtividade, redução de erros operacionais e melhoria na qualidade dos serviços prestados.
